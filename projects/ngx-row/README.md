# NgxRow

## Demo

[https://liangshen001.github.io/ngx-row-lib/](https://liangshen001.github.io/ngx-row-lib/)

## Install
````
npm i ngx-row -s
````

## Usage

### import module

```
import {NgxRowModule} from "ngx-row";

@NgModule({
  imports: [
    NgxRowModule,
    ...
  ],
  ...
})
export class AppModule { }
```
